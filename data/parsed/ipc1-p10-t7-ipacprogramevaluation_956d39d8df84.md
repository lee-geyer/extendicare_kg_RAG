|Definition/Description:| | | | | | | |
|Infection:| | | | | | | |
|• This is the entry and multiplication of an infectious agent in the tissues of the host.| | | | | | | |
|There are two types of infection: asymptomatic and symptomatic.| | | | | | | |
|• Asymptomatic or sub-clinical infection is an infectious process running a course similar to that of clinical| | | | | | | |
|disease but below the threshold of clinical symptoms.| | | | | | | |
|• Symptomatic or clinical infection is one resulting in clinical signs and symptoms i.e. disease.| | | | | | | |
| | | | | | | | |
|Outbreak:| | | | | | | |
|This is an increase in the number of cases (colonization or infections) above the number normally occurring in a| | | | | | | |
