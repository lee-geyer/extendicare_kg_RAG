# Extendicare Maintenance Inspection Checklist

## Equipment
### Expansion Compensation Tanks

### LEGEND/KEY
- **W** – Weekly
- **M** – Monthly
- **Q** – Quarterly
- **S** – Semi-Annual
- **A** – Annual
- **O** – Ok
- **R** – Repair
- **N** – Replace

### Item Checks
| W | M | Q | S | A | Item                                                                 |
|---|---|---|---|---|----------------------------------------------------------------------|
| X |   |   |   | 1 | Check tank and connections for leaks.                                |

|   | X |   |   | 4 | Check system pressure. Recharge if necessary.                       |
